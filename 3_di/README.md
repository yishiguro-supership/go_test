# 3_di

## Description
- DI(Dependency Injection)を行うためのwireについての検証
  - 3-1_no_di：DIを行わず、モジュールに依存したコード
  - 3-2_di_without_wire：wireを使わずに自力でDIするコード
  - 3-3_di_with_wire：wireを使ってDIするコード
